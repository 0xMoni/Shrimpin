# Quick Installation Guide

## 1. Start the Backend

```bash
cd ~/Shrimpin
python3 posture_detector.py
```

Leave this running. You should see:
```
Backend running on http://localhost:8765
```

## 2. Load the Extension

1. Open Chrome
2. Go to `chrome://extensions/`
3. Enable **Developer mode** (top-right toggle)
4. Click **Load unpacked**
5. Select folder: `/Users/moni/Shrimpin/extension`

## 3. Activate

1. Click the 🦐 extension icon in your toolbar
2. Toggle **Detection** to ON
3. The widget appears on all webpages!

## 4. Test It

1. Visit any website (e.g., google.com)
2. You should see the floating shrimp widget
3. Sit up straight for calibration
4. Try slouching to trigger an alert

---

**That's it! You're ready to stop shrimping! 🦐**
