🦐 Shrimpin' Extension Test Page

Use this page to test if the extension is working

📋 Test Steps

  1. Open DevTools (Cmd+Option+J or F12)
  2. Go to the Console tab
  3. Click the 🦐 extension icon in your toolbar
  4. Toggle "Detection" to ON
  5. Watch the console for messages starting with 🦐
  6. Look for the floating widget in the bottom-right corner

✅ What You Should See

In Console:

🦐 Widget injected
🦐 Widget initialized
🦐 Loaded settings: {settings: {...}}

🐛 Troubleshooting

Widget not appearing?

  • Refresh this page (Cmd+R)
  • Make sure extension is loaded at chrome://extensions
  • Check console for errors (red text)

No console messages?

  • Go to chrome://extensions
  • Click refresh icon on Shrimpin' extension
  • Reload this page

Toggle doesn't work?

  • Right-click the popup → Inspect
  • Check popup console for errors
  • Look for "Toggle clicked, enabled: true"

🔍 Manual Test Commands

Run these in the console to test manually:

Check if content script loaded:

console.log('Widget exists?', !!document.getElementById('shrimpin-root'));

Check current settings:

chrome.storage.local.get(['settings'], (result) => {
  console.log('Current settings:', result);
});

Manually start detection:

chrome.runtime.sendMessage({ action: 'startDetection' }, (response) => {
  console.log('Response:', response);
});

Manually toggle ON via storage:

chrome.storage.local.set({
  settings: { enabled: true, volume: 0.5, soundEnabled: true }
}, () => {
  console.log('Settings updated!');
});
